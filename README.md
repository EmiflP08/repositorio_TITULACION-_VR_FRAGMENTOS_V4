# repositorio_TITULACION-_VR_FRAGMENTOS
Repositorio para el proyecto de titulación del eje interactivo, realizado por Andrea Imaicela y Emilia Pérez. 2026.

### README 
Experiencia VR: Fragmentos Perdidos 

## Descripción del proyecto 

Este proyecto es una experiencia inmersiva en realidad virtual desarrollada en Unity que busca generar reflexión sobre las consecuencias físicas y cognitivas de la mala calidad del sueño en jóvenes y adultos. 

A través de un recorrido interactivo en primera persona, el usuario experimenta efectos de la mala calidad del sueño, representados mediante entornos, interacciones y estímulos sensoriales. 
La experiencia culmina en un espacio de reflexión que refuerza el mensaje central: 
“Dormir también es cuidarte, cada hora de sueño cuenta.” 

 
## ¿Por qué es útil este proyecto? 

La mala calidad del sueño es un problema común, pero muchas veces sus consecuencias no se perciben de forma inmediata. 

Este proyecto propone una forma experiencial de abordar el tema: en lugar de solo informar, permite que el usuario viva los efectos del mal descanso, fomentando una mayor reflexión sobre sus hábitos. 

 

## Características principales 
- Experiencia VR en primera persona
- Sistema de locomoción híbrido: teleport + movimiento libre 
- Interacción con objetos del entorno
- Narrativa guiada mediante voz 
- Representación de consecuencias de la mala calidad del sueño 
- Sistema simbólico de fragmentos (recolección y reconstrucción final) 
 
## Cómo ejecutar el proyecto 

**Requisitos:**
- Unity 6 (6000.x LTS) 
- Meta Quest 2 (o dispositivo compatible)  
- XR Interaction Toolkit (configurado en el proyecto) 
- Cable para conectar el casco con la computadora

**Pasos:**
- Clonar el repositorio: 
https://github.com/EmiflP08/repositorio_TITULACION-_VR_FRAGMENTOS.git

- Abrir el proyecto en Unity Hub 
- Verificar configuración XR (XR Interaction Toolkit activo) 
- Conectar el visor Meta Quest  
- Ejecutar la escena principal desde Unity 

## Controles
- **Movimiento libre:** Joystick
- **Teleport:** Trigger (gatillo)
- **Agarrar objetos (grab):** Grip button
